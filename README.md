# swc-nodemon
Basic example of swc with nodemon

Getting Started
------------
  * Clone
    ```
    git clone https://github.com/aelbore/swc-nodemon
    cd swc-nodemon
    ```
  * Install dependencies
    ```
    yarn install
    ```
  * Run the Application
    ```
    yarn serve
    ```
  * Open browser then go to `http://localhost:3000/api/users`
